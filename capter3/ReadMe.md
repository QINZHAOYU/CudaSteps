## 单源文件 CUDA 程序基本框架

对于单源文件的 cuda 程序，基本框架为：

    包含头文件

    定义常量或宏

    声明 c++ 自定义函数和 cuda 核函数的原型

    int main()
    {
        分配主机和设备内存
        初始化主机中数据
        将某些数据从主机复制到设备
        调用核函数在设备中计算
        将某些数据从设备复制到主机
        释放主机和设备内存
    }
